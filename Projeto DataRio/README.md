# Estudo sobre indicadores econômicos do estado do Rio de Janeiro (2000 - 2006)

Projeto de análise de indicadores econômicos do estado do Rio de Janeiro utilizando dados distribuídos pelo DataRio e fornecidos pelo IBGE.
  
Os dados foram fornecidos contemplavam diversos indicadores econômicos.  
Dos dados disponibilizados pelo Data Rio, este trabalho utilizou para limpeza, análise e inferência os indicadores de: 
<strong>
Receita Total | Salários | Número de Empresas | Pessoal Ocupado
</strong>.
